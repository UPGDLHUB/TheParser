# Parser

A draft for a **Java-based predictive recursive-descent parser** that processes a  
sequence of tokens generated by a lexer and validates the syntactic  
structure of a simple programming language.

Currently, the parser supports:

-   class declarations
-   global variable declarations
-   optional assignments
-   simple literal expressions

It reads the **token vector produced by the lexer** and verifies
that the input program follows the defined grammar.

# Grammar Overview

The parser currently supports a simplified grammar similar to:

    clase               -> "class" IDENTIFIER "{" declaracion_variable* "}"

    declaracion_variable
                        -> tipo IDENTIFIER ("=" expresion)? ";"

    tipo                -> entero | int | char | cadena | string | booleano | boolean

    expresion           -> literal

    literal             -> INTEGER | CHARACTER | STRING

# Running the Parser

1.  The lexer scans the input file and produces tokens.
2.  The parser reads the token vector.
3.  The parser validates the syntactic structure.

Example output:

    rule_clase (root)
    -rule_declaration_variable
    -rule_expression
    -rule_literal
    Parsing completed successfully.
